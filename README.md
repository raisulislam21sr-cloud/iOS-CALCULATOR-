# iOS-CALCULATOR-

This repository contains the Swift code for an iOS Calculator and Clock app.

## Features
- FULLY FUNCTIONAL IOS CALCULATOR REPLICA
- WITH SAME UI

## How to Run
1. Copy the Swift files into an Xcode project
2. Open the project in Xcode
3. Run on iPhone simulator or device

## Tech Used
- Swift 5
- Xcode (iOS/macOS)
